# 🚀 ChronoLearn
Explore History. Understand Science. Share Knowledge.
ChronoLearn is an interactive educational platform that combines history, science evolution, and collaborative research sharing through immersive animations and structured knowledge systems.

This project was developed with the assistance of AI tools. I focused on designing the concept, structure, and functionality, while using AI to help with implementation and code generation. I used AI as a tool to build this concept faster, and focused on designing the system and user experience.
